# Readiness Review

## Process

1. Load the rubric from `data/rubrics.json`.
2. Score each dimension red, yellow, or green.
3. Ground findings in evidence from the user's artifacts and connected sources.
4. Produce fixes that can be completed before the deadline.

## Output

- readiness scorecard
- critical blockers
- quick wins
- missing evidence
- revised next actions
