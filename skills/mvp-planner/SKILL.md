# MVP Planner

## Process

1. Define the user outcome.
2. Separate must-have from later features.
3. Pick the simplest build path: no-code, web app, AI workflow, data notebook, web3 app, or manual concierge.
4. Produce a timeline based on team and deadline.
5. Identify dependencies and credentials.

## Output

- MVP promise
- non-goals
- feature list
- build path
- timeline
- risks
- demo script
